Free Open Source Licenses

一直以来都觉得 JetBrains 系的产品是我入门以来用过最好的 IDE，工作两年后一直想买套支持一下。

但毕竟囊中羞涩，动辄一两百刀的价格实在是让人望而却步……

其实社区版已经足够良心了，所以在拥有 Licenses 之前，还是劝大家使用社区版。

直到有一天我找到了一个开源项目支持计划：

总的来说，除了项目是属于你本人的、持续开发超过 3 个月、保持定期更新之外，没什么硬性要求。

star 数什么的也完全完全完全不 care~

要项目呢，我也不是没有：https://github.com/yipwinghong/algorithm

虽然不是什么高端玩意，但还是保持了三四个月的活跃度，用来过审是绰绰有余的了：

因此，只需要在 Open Source License Request 填写项目的基本信息：

项目名称，主要的开发语言，维护时间，网站、仓库、许可证的地址，国家和地区，需要申请的许可证数量，以及其他的个人信息。

提交申请后您的邮箱就会收到这样的一封邮件：

然后等待审核回复就可以了，在一周后我就收到了邮件：

就这样，我拿到了许可证咯：

  • 有效期为一年，到期前会收到邮件提醒。如果到时候您的项目仍然符合要求，可以继续 申请续期；
  • 许可证 仅限用于非商业开源项目，如果您获得多于 1 个许可证，可以与项目的其他活跃贡献者分享；
  • 如果认为 JetBrains 的软件给您的项目带来很大的帮助，希望可以在您的项目主页中提到 JetBrains 的支持，还可以把 JetBrains 的 Logo 和链接放在您的开源项目中。

需要注意的是，持有此许可证一年内 软件可免费升级到所有最新版本，允许安装到任意台主机上，但不能同时在两台或以上的主机上使用，且使用权只属于授权用户，无权转让给任何第三方。

在进入邮件中提供的链接，会提示使用之前收到邮件的邮箱注册 JetBrains 账号，激活后即可在您的 JetBrains 个人主页 Licenses 一栏看到全家桶工具箱的订阅。

选择 Download activation code for offline usage，即可下载激活码离线使用。

还可以使用 JetBrains 账号登录，一样的。

也就是说，AppCode、CLion、Intelli IDEA、WebStorm 等宇宙最强系列 IDE 都可以免费使用了。

所以赶紧下载 ToolBox，把我的社区版全部换掉。在注册软件后可见：
